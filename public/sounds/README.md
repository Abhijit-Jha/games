# Sound Effects

Add the following MP3 sound files to this directory:

- `click.mp3` - UI click sound
- `score.mp3` - Score increment sound
- `game-over.mp3` - Game over sound
- `jump.mp3` - Jump action sound
- `type.mp3` - Typing keystroke sound
- `success.mp3` - Success/completion sound
- `error.mp3` - Error/mistake sound

Recommended: Use short, subtle sound effects (under 1 second each).
Free sound effects can be found at:
- https://freesound.org
- https://mixkit.co/free-sound-effects/
